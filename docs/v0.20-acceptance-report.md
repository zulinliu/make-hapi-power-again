## 四、关键设计决策

1. **双令牌体系保留**: `--hp-*` (foundation) + `--app-*` (semantic alias)，Phase 5 仅在 SessionChat 中将直接使用的 `--app-*` 替换为 `--hp-*`，语义别名层仍保留
2. **DataBoundary early return**: 对 nullable data 采用 early return + 独立 skeleton/error 渲染，避免 TypeScript 泛型推断与 null 不兼容
3. **ActionMenu 双形态**: 桌面 Popover + 移动 Bottom Sheet，通过 AdaptiveProvider 自动切换
4. **OverlaySurface 移动映射**: popover/context-menu → bottom-sheet, command-palette → full-screen-sheet
5. **FileManager 内部组件保持独立**: 其 Dialog/ContextMenu/Toast 是功能等效但内部使用的组件，未强行替换以避免破坏性回归
