## 发布前检查

- 所有截图不得包含真实 API key、token、私有 Git remote credential、个人绝对路径或真实内部 host。
- 如需要演示敏感字段，使用 `example.com`、`git.internal.example.com`、`test-user`、`/home/tester/project` 等占位值。
- 截图应覆盖浅色/暗色至少一种主视觉，移动端必须验证 safe-area、键盘和底部操作区不重叠。
- README 替换截图前先确认文件存在且体积合理，避免仓库引入过大的临时导出。
