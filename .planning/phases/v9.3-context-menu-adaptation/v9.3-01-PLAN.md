# Plan v9.3-01: useContextMenu Hook + Git Status Panel Context Menu

> Create the `useContextMenu` hook and add right-click + "..." button context menu to GitStatusPanel file rows.

## Goal

Add right-click context menu support to Git status panel file rows with a reusable `useContextMenu` hook that wraps `useLongPress`. The menu provides View Diff, Copy Path, and Open File actions. Each file row also gets a visible "..." button for mobile-friendly access.

## Context

- Decisions: D-01 (useContextMenu hook), D-02 (Git menu = View Diff + Copy Path + Open File), D-03 (Git file rows get "..." button)
- Requirements: CTXMENU-01, CTXMENU-05, CTXMENU-06
- Research: `v9.3-RESEARCH.md` — verified all API contracts, import paths, and pitfalls from source code
- CTXMENU-02/03/04 already handled (DirectoryTree has right-click, ContextMenu uses `--app-*`)
- Key reference: `files.tsx` lines 237-244 (contextMenu state), 272-285 (handleOpenFile navigation), 368-377 (menu handler)
- Pitfall: Need `justClosedRef` pattern to prevent menu re-opening on touch dismiss (RESEARCH Pitfall 2)
