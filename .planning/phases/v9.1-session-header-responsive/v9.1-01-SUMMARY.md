## Key Changes

### Task 1: Shared SessionHeader with Route-Aware Navigation

- **SessionDetailRoute** now wraps all child routes with a shared `SessionHeader`, rendering it unconditionally above `{isChat ? <SessionPage /> : <Outlet />}`
- **SessionHeader** props simplified: removed 8 callback props (`onViewFiles`, `onViewGit`, etc.), added `sessionId`, `isSubPage`, `onToggleOutline`
- Uses `useLocation()` and `useNavigate()` from TanStack Router internally for route detection and toggle navigation
- Active icon highlighted with `text-[var(--app-link)]` when route matches that tool
- Header shows `border-b` only when `isSubPage` is true (sub-pages visible, not on chat)
- **SessionChat** no longer renders its own SessionHeader; accepts `outlineOpen`/`onOutlineOpenChange` as props
- Outline state flows: `SessionDetailRoute` -> `SessionHeader` (toggle) + `SessionPage` -> `SessionChat` (render overlay)

### Task 2: Responsive Menu + Header Removal

- **SessionActionMenu** hides Tools section on desktop (>=1024px) using `matchMedia`, keeps on mobile
- Session management (Rename, Archive/Delete) always visible in menu
- All 8 sub-pages have their independent headers removed:
  - git.tsx: action buttons moved to content toolbar, tab bar preserved
  - extensions.tsx: tab bar preserved
  - changes.tsx: filter tabs preserved
  - timeline.tsx: checkpoint button relocated to filter bar
  - undo.tsx: scope selector starts immediately
  - files.tsx: refresh button merged into search bar area, tab bar preserved
  - file.tsx: file path bar preserved
  - terminal.tsx: ConnectionIndicator relocated as absolute overlay
- Git page tab bar switched from `--hp-*` to `--app-*` CSS variables
- All `pt-[env(safe-area-inset-top)]` removed from sub-pages (handled by shared SessionHeader)
