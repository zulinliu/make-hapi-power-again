## 预览触发方式

| Option | Description | Selected |
|--------|-------------|----------|
| 单击文件行直接预览 | 单击直接在 Git 页面内打开预览面板，右键菜单 View Diff 仍导航到 file 路由 | ✓ |
| 仅通过右键菜单 | 保持现有行为不变，仅通过右键/··· 菜单新增「预览」项 | |
| 单击 + 右键菜单都有 | 单击触发内联预览，右键菜单 View Diff 改为也触发内联预览 | |

**User's choice:** 单击文件行直接预览
**Notes:** 最直观的体验，右键菜单行为保持不变
